Ravencoin raven-qt Special Interest Group
*A subgroup of the Ravencoin Code Committee*


Here you will find information about the Ravencoin raven-qt Special Interest Group

The raven-qt SIG is a subgroup of the Ravencoin Code Committee

For general information about Ravencoin, please go to https://raven.wiki/

Latest News:

  • The Ravencoin raven-qt node software is currently undergoing testing for P2SH assets, which enables the use of Multisig and Atomic_Swaps for assets. This new feature will be a chain hard fork.
  • Please note that the Ravencoin testnet has already forked for P2SH assets, but mainnet has not yet forked.
  • For this reason:
    • The "official" latest release for mainnet is still V4.3.2.1 at https://github.com/RavenProject/Ravencoin/releases/tag/v4.3.2.1
    • If you would like to test a newer version of raven-qt on mainnet, please build from the "master" branch of https://github.com/RavenProject/Ravencoin. That branch has most new features, but not P2SH assets.
    • If you would like to help test P2SH assets code on testnet (please do!!), then please use https://github.com/Ravenqt-RVN-SIG/Ravencoin/releases/tag/v4.7.0test1
    • If you want to test the very latest code on testnet (including P2SH), then please build from the "develop" branch of https://github.com/RavenProject/Ravencoin.


The Ravencoin raven-qt Special Interest Group is focused on generating testnet binaries, debugging problems which occur, developing new code, and making recommendations for future mainnet releases.

Please note that all software here are WORKS IN PROGRESS.
You are welcome to use any software or other information posted here, but you do so AT YOUR OWN RISK.

Most communications within the Ravencoin raven-qt SIG takes place on the Ravencoin Discord server.
  • Real-time work takes place in the #ravenqt-sig-working text channel. Anyone can participate,
  • Meetings take place in the #ravenqt-sig-meetings voice channel weekly on Wednesdays at 19:30 UTC. Anyone can listen to the SIG meetings but only SIG members can speak.
  • Links to presentations are posted in the #ravenqt-sig-meetings  text channel.

For questions or to volunteer to help with the SIG activities, please post in the #ravenqt-sig-working Discord text channel.
