Ravencoin raven-qt Special Interest Group
*A subgroup of the Ravencoin Code Committee*


Here you will find information about the Ravencoin raven-qt Special Interest Group

The raven-qt SIG is a subgroup of the Ravencoin Code Committee

For general information about Ravencoin, please go to https://raven.wiki/


The Ravencoin raven-qt Special Interest Group is focused on generating testnet binaries, debugging problems which occur, developing new code, and making recommendations for future mainnet releases.

Please note that all software here are WORKS IN PROGRESS.
You are welcome to use any software or other information posted here, but you do so AT YOUR OWN RISK.

Most comunications within the Ravencoin raven-qt SIG takes place on the Ravencoin Discord server.
  • Real-time work takes place in the #ravenqt-sig-working text channel. Anyone can participate,
  • Meetings take place in the #ravenqt-sig-meetings voice channel (weekly time slot to be announced). Anyone can listen to the SIG meetings but only SIG members can speak.
  • Links to presentations are posted in the #ravenqt-sig-meetings  text channel.

For questions or to volunteer to help with the SIG activities, please post in the #ravenqt-sig-working Discord text channel.
